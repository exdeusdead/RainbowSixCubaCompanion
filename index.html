Rainbow Six CUBA Companion

The Companion extension helps synchronize public Rainbow Six Siege statistics from Tracker Network with the Rainbow Six CUBA Stats bot.

It does not ask for Ubisoft credentials, Discord passwords, payment information or private messages.

Install through Discord flow
Chrome Web Store link will be added after approval.
Install the extensionUsers install Rainbow Six CUBA Companion once it becomes available.
Connect UbisoftThe Discord bot generates a Tracker Network profile link for the user.
Open R6 TrackerThe extension reads public profile information from the opened Tracker page.
Sync to R6 StatsStats become available for profiles, rankings and competitive panels inside the server.

Privacy-first by design.

The system is built around public competitive information and clear user action. No Ubisoft login is required for synchronization.

What is used

  • Public game statistics
